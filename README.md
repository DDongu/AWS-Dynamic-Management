# Setting

### 1. 필요 모듈설치

```
$ pip install boto3
```

### 2. 파일 세팅

**Windows)** _C:\Users\\[사용자명]\\.aws\credentials_

```
[default]
aws_access_key_id = YOUR_ACCESS_KEY
aws_secret_access_key = YOUR_SECRET_KEY
```

**Windows)** _C:\Users\\[사용자명]\\.aws\config_

```
[default]
region=us-east-1
```
